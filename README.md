## 🚀 Quick Start

### Option 1: Double-click Launch (Recommended)
1. Double-click `Launch-Portal.bat`
2. Wait for your browser to open automatically
3. Explore the programs!

### Option 2: Manual Launch
1. Open Command Prompt or PowerShell
2. Navigate to the portal directory
3. Run: `python server.py`
4. Open http://localhost:8080 in your browser

## 🔍 Troubleshooting

### Server Won't Start
- Check if Python is installed: `python --version`
- Try a different port if 8080 is busy
- Run as administrator if needed

### Browser Doesn't Open
- Manually open http://localhost:8080
- Check firewall settings
- Try a different browser

### Code Not Loading
- Check internet connection
- GitHub repository must be public
- Files must exist in the repository
